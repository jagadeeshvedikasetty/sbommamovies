üìê Image Specifications

  • Format: JPG or PNG (JPG recommended for smaller file size)
  • Aspect Ratio: 2:3 (Portrait) - e.g., 300x450 or 400x600
  • Recommended Dimensions: 300x450 pixels or 400x600 pixels
  • File Size: Keep under 200KB for faster loading
  • Naming: Use lowercase with hyphens (e.g., my-movie-name.jpg)

üîß Troubleshooting

‚ö†Ô∏è Images not showing?
  • Check the file path is correct (case-sensitive!)
  • Make sure the image file exists in the folder
  • Open browser console (F12) and check for errors
  • Verify file extension matches (.jpg vs .jpeg)
  • For React: Make sure the path starts with / (e.g., /images/...)
  • For HTML: Use relative path (e.g., images/...)
‚ö†Ô∏è Images loading slowly?
  • Compress your images (use TinyPNG or similar)
  • Reduce image dimensions to recommended size
  • Convert PNG to JPG if possible
